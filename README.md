# Šķidruma balansa kalkulators

Vienkārša HTML/PWA aplikācija šķidruma balansa aprēķinam.

## GitHub Pages palaišana

1. Izveido GitHub repozitoriju, piemēram, `skidruma-balansa-kalkulators`.
2. Augšupielādē šos failus repozitorija saknē:
   - `index.html`
   - `manifest.json`
   - `service-worker.js`
   - `icon-192.png`
   - `icon-512.png`
3. GitHub repozitorijā atver **Settings → Pages**.
4. Pie **Build and deployment** izvēlies:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Saglabā. Pēc dažām minūtēm aplikācija būs pieejama GitHub Pages saitē.
